# Google-Dashboard-Clone 🔍
Primer pagina recreada de Google

Una recreación de la página principal del buscador de Google, desarrollada como práctica de maquetación web frontend. Este proyecto es un clon estático y responsivo que imita la interfaz de usuario clásica del buscador.

## 🚀 Tecnologías utilizadas

* **HTML5:** Estructuración semántica del contenido.
* **CSS3:** Estilos visuales, uso de Flexbox para el centrado absoluto de elementos y *Media Queries* para garantizar la adaptabilidad en dispositivos móviles.
* **JavaScript (Vanilla):** Lógica básica para capturar datos en la barra de búsqueda y disparar alertas simulando la interacción del usuario.

## 🛠️ Cómo ejecutar el proyecto

Dado que es un proyecto frontend estático puro, no requiere la instalación de dependencias ni levantar un servidor local. Para verlo en acción, solo sigue estos pasos:

1. Descarga los archivos en tu computadora asegurándote de que `index.html`, `style.css` y `script.js` estén en la misma carpeta.
2. Haz doble clic sobre el archivo **`index.html`**.
3. El proyecto se abrirá automáticamente en tu navegador web predeterminado (Chrome, Firefox, Edge, etc.) y estará listo para usarse.

## 💡 Características

* Diseño limpio y minimalista fiel al original.
* Interfaz responsiva que se adapta a pantallas de escritorio, tablets y móviles.
* Simulación de eventos de búsqueda al hacer clic en el botón "Buscar en Google" o al presionar la tecla `Enter`.
